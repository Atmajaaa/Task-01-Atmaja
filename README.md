Project Title: Data Cleaning and Preparation

Objective:
To clean and prepare an online store dataset for analysis by identifying missing values, checking duplicate records, and validating data formats.

Tools Used:

* Python
* Pandas
* Microsoft Excel

Steps Performed:

1. Loaded the dataset using Pandas.
2. Examined dataset structure and dimensions.
3. Identified missing values in each column.
4. Found 309 missing values in the CouponCode column.
5. Replaced missing CouponCode values with "No Coupon".
6. Checked for duplicate records and found none.
7. Verified data types for all columns.
8. Exported the cleaned dataset to a new Excel file.

Results:

* Total Records: 1200
* Total Columns: 14
* Missing Values Found: 309 (CouponCode)
* Duplicate Records: 0
* Dataset successfully cleaned and prepared for further analysis.
